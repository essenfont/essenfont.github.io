# 18 — Build pipeline

The current pipeline:

```
scripts/gen-ssg-routes.mjs → public/ssg-routes.json + public/sitemap.xml
vite-ssg build             → dist/
```

The Astro pipeline:

```
scripts/gen-unicode-data.mjs → public/unicode-blocks.json + per-block JSON + indexes
scripts/gen-search-index.mjs → public/search-index.json    (new — step 12)
astro build                   → dist/                       (replaces vite-ssg)
@astrojs/sitemap              → dist/sitemap-index.xml      (replaces hand-rolled sitemap.xml)
```

## npm scripts

```json
{
  "scripts": {
    "dev": "astro dev",
    "dev:legacy": "vite",
    "build": "npm run gen-search-index && astro build",
    "build:legacy": "node scripts/gen-ssg-routes.mjs && vite-ssg build",
    "preview": "astro preview",
    "gen-unicode": "node scripts/gen-unicode-data.mjs",
    "gen-search-index": "node scripts/gen-search-index.mjs",
    "test": "node --test tests/*.test.mjs"
  }
}
```

Keep `build:legacy` until step 21 (decommission).

## Acceptance

- `npm run build` produces `dist/` with same route count as legacy build
- `dist/sitemap-index.xml` generated by Astro matches (or improves) the legacy sitemap
- Build time is within 2× the legacy build (~10 min for ~160k pages)
